● はじめに

	▼ 概要

		FlacBirdは音楽ファイルのタグを自動で取得するソフトです。タグの情報はiTunes等
		でも使用されている信頼性の高いデータベースであるGracenoteから取得します。

		タグの自動取得ソフトは多くありますしGracenoteを使用したものも少しはありますが
		曲ごとに取得するタイプのものがほとんどで、本当に正確なタグ情報を取得するため
		にはiTunesのようにアルバム単位での取得が必要になります。それはつまりCDを挿入
		し、アルバム情報を取得してから音楽ファイルに変換する必要があるということです。

		では既にmp3などの圧縮ファイルに変換済みのデータのタグを取得するにはどうすれば
		よいのでしょうか？もちろん曲ごとにタグを取得する手法もGracenote MusicIDなどの
		技術により精度は上がってきています。しかしながら単曲での識別はなかなか困難で
		あり、1枚のアルバムがあったとして曲ごとにタグを取得すると、シングルカットされ
		た曲などではアルバム名を間違えたり似ている曲のタグを誤って取得するケースも多
		くあります。

		FlacBirdでは複数の音楽ファイルをひとつのアルバムとしてTOC情報を生成し、その
		データをもとにGracenoteからアルバム単位でのタグを取得します。これによりひと
		つのアルバム中で曲名を誤って取得することが無くなりますのでiTunesとほぼ同じ
		精度のタグを取得することができるようになります。

● 注意事項

		タグやアートワークを取得する際に使用するAPIは下記の情報を送信します。

	▼ Gracenote Web API

		タグを取得する際に使用します。曲の長さ(TOC)を送信します。
		https://developer.gracenote.com/web-api

	▼ iTunes Store Web Service Search API

		アートワークを取得する際に使用します。アルバム名とアーティスト名を送信します。
		https://www.apple.com/itunes/affiliates/resources/documentation/itunes-store-web-service-search-api.html

	▼ Last.fm Web Services

		アートワークを取得する際に使用します。アルバム名とアーティスト名を送信します。
		http://www.lastfm.jp/api/intro

● 主な機能

	▼ アルバム単位でのタグ情報の取得

		フォルダに含まれるファイルをひとつのアルバムとしてタグ情報を取得します。
		(単曲でのタグ情報取得には対応していません)

	▼ アートワークの取得

		アーティスト名、アルバム名をもとにアートワークを取得します。画像ファイル
		はタグに書き込むことも、フォルダにファイルとして出力することもできます。

	▼ フォルダの整理

		iTunesの「フォルダを整理」に似た機能です。タグ情報をもとにアーティスト名やア
		ルバム名など指定した形式でファイル名とフォルダ構成を整理します。

● 対応形式

		Advanced Audio Coding (aac)
		Monkey's Audio (ape)
		Free Lossless Audio Codec (flac)
		MPEG-4 (m4a, mp4)
		MPEG Audio Layer-3 (mp3)
		Ogg Vorbis (ogg)
		Windows Media Audio (wma)
		RIFF waveform Audio Format (wav)
		Musepack (mpc)
		WavPack (wv)
		Cue sheet (cue)

● 動作環境

		Windows XP, Vista, 7, 8 (32ビット)
		Windows 7 (64ビット) ※ 32ビットモードで動作します。

● インストール

		アーカイブを適当なフォルダに解凍します。

● アップデート

		解凍した中身をすべて上書きしてください。

● アンインストール

		レジストリは使用していませんので、解凍したフォルダを削除してください。
		設定ファイルは下記のフォルダに保存されています。

			Windows XP
				C:\Documents and Settings\xxxx\Application Data\FlacBird

			Windows Vista, 7, 8
				C:\Users\xxxx\AppData\Roaming\FlacBird

● 使い方

	(1) 準備

		事前にmp3などの音楽ファイルをアルバムごとにフォルダわけしておきます。この時
		ファイル名の先頭に数字を入れるなどでアルバムの曲順に並ぶようにしておいてくだ
		さい。

	(2) ファイルの追加

		FlacBirdを起動すると空のリストが表示されていますので、先ほどのフォルダをドラ
		ッグドロップするか「ファイル」メニューの「ファイルの追加」または「フォルダの
		追加」を選択することでリストに対象のファイルが追加されます。

		既にfreedbなどでタグにトラック番号が入っている場合はそのトラック番号を優先、
		タグが無い場合はファイル名の順に追加されます。

	(3) すべてのタグを取得

		「詳細」メニューから「トラック名の取得」を選択するとリストに登録されているす
		べてのファイルのタグ情報を取得します。この時、複数一致するものが見つかった場
		合は最善だと思われるものを自動的に取得します。

		取得できなかったファイルは灰色、データベースに複数一致するものが見つかった場
		合は赤色、完全に一致した場合は青色になりチェックが入ります。

	(4) 赤色の項目の修正

		データベース上に複数一致するものが見つかると赤色の表示になります。赤色に表示
		されている項目をダブルクリックすると一致する複数の項目が一覧で表示されます。
		一覧から適切なアルバム名を選択してください。

	(5) タグの書き込み

		完全に一致したファイルは青色になり、自動的にチェックが入ります。タグの書き込
		み対象はチェックが入っているファイルのみです。赤色の項目でもタグに誤りが無け
		れば手動でチェックを入れることで書き込み対象になります。

		ツールバーの保存ボタンか「ファイル」メニューの「書き込み」を選択するとチェッ
		クの入っているファイルに対してタグを書き込みます。

	(6) フォルダの整理

		フォルダの整理はタグの取得とは完全に別の機能です。先ほどタグの取得で一覧に登
		録したリストは関係ありません。また、タグ情報が正常に入っていることが前提です。

		「詳細」メニューの「フォルダを整理」を選択します。

		空のリストが表示されますのでタグが書き込まれたファイルをドラッグドロップする
		か「追加」ボタンを押してファイルを追加します。

		「形式」を好みに合わせて指定します。「>」をクリックすることで使用できる項目
		を一覧から選択することも可能です。初期状態では iTunes の形式に合わせた値が入
		力されています。

		「出力先」で出力先のフォルダを指定します。

		プレビューでフォルダ構成とファイル名を確認し、問題が無いようであれば「OK」を
		クリックすることで出力先のフォルダに対象のファイルを移動します。

● 制限事項

	(1) 作曲者(Composer)はGracenote Web APIにないため取得できません

	(2) ディスク番号/ディスク枚数はデータがありませんので取得できません

	(3) iTunesは独自のデータを使用している場合がありますので完全にiTunesのタグと
		一致するわけではありません
